Check if a variable is null or undefined in JavaScript

We can use strict equality to check if a variable is null or undefined in JavaScript. if (value === null || value === undefined) { // value is either null or undefined } What I also found interesting is that although the regular equality in JavaScript is generally frowned upon, these two conditions are exactly identical. Of course, in general I would still prefer to use the code above as it is explicit in its intent.

Check the status of a Git repository

To check the status of the Git repository. git status We can see a --short or -s version with this command. git status --short For convenience, you can add git st as an alias. git config --global alias.st "status --short"
