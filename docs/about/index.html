About

This is not a blog.

These notes are written for myself as a reference and mostly center around software development, but may occasionally include other topics. They are deliberately concise. Think of it as a brain archive similar to Josh Branchaud’s Today I Learned.

The easiest way to search for a note is to view them in one page. They are organized by tags and categories.

These notes are written in Markdown, built into a static site with Hugo, and hosted on GitHub Pages.
