Buying a Raspberry Pi

There are many different models of Raspberry Pi depending on your needs.

Once you have decided which model you want, the easiest option is to purchase a CanaKit Raspberry Pi starter kit. It comes with all the essential accessories you will need to get started including a case with a fan, some heat sinks, and a power supply. Try searching on Amazon for the best deals for Raspberry Pi kits.
