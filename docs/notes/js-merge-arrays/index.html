Merging arrays in JavaScript

There is no straightforward way to merge two arrays in JavaScript.

const a = [
  'apple',
  'banana',
  'cherry',
]

const b = [
  'cherry',
  'durian',
  'elderberry',
  'fig'
]

const c = [...a, ...b]

The problem with this method is that it won’t work for arrays with objects.

const a = [
  {
    name: 'Bob',
    country: 'Singapore'
  }
]

const b = [
  {
    name: 'Bob',
    country: 'Malaysia'
  }
]

const c = [...a, ...b]

// result is c now contains duplicates 
[
  {
    name: 'Bob',
    country: 'Singapore'
  },
  {
    name: 'Bob',
    country: 'Malaysia'
  }
]
